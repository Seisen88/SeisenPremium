# Seisen Hub Premium

The official premium dashboard and website for Seisen Hub, a high-quality Roblox script hub.

## Features

- **Premium Dashboard**: Secure client area for managing subscriptions.
- **Automated Payments**: Integration with PayPal and Robux payment gateways.
- **Instant Delivery**: Automatic license key generation and delivery.
- **Support System**: Integrated ticket system for customer support.
- **Admin Panel**: Comprehensive tools for managing users, tickets, and payments.

## Tech Stack

- **Frontend**: Next.js 14 (React), Tailwind CSS
- **Backend**: Next.js API Routes
- **Database**: Supabase
- **Authentication**: Custom Auth (Supabase) + JWT

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Set up environment variables (`.env.local`).
4.  Run the development server: `npm run dev`
